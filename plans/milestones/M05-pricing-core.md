# M05: 価格改定コア機能

**Status**: pending
**Depends on**: M04

## 概要
CSV で新価格を投入 → セット商品展開 → 価格入力 → プレビュー → 実行 の中核フロー。

## 実装内容
- 価格改定 CSV テンプレートダウンロード（`syohin_code,new_price` 形式）
- 価格改定 CSV アップロード
- `syohin_code` → 関連 `set_syohin_code` 自動洗い出し
- セット商品価格入力 UI（テーブル形式）
- 税込価格自動計算（Yahoo/Shopify 向け、`floor()` 切り捨て）
- プレビュー画面（現在価格 → 新価格、モール別 × 商品別一覧）
- 反映対象モール選択チェックボックス
- `price_change_run` 作成 + `price_change_log` 記録開始

## 受入基準
- [ ] CSV アップロードで新価格を投入できる
- [ ] 関連セット商品が自動的に表示される
- [ ] セット商品に個別価格を入力できる
- [ ] プレビューに「現在価格 → 新価格」が全モール分表示される
- [ ] 反映対象モールを選択できる
- [ ] 税込計算が正しい（`floor(税抜 × (1 + tax_rate/100))`）
