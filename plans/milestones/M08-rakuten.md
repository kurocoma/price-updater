# M08: API連携 — 楽天 RMS

**Status**: done
**Depends on**: M05

## 概要
楽天 ItemAPI で商品価格を更新する。

## 実装内容
- ItemAPI（item.update / items.update）による価格更新
- キー: SKU管理番号（またはシステム連携用SKU番号）
- 値: 通常購入販売価格（税抜）
- himoduke による商品コード変換
- セット商品の価格更新
- エラーハンドリング & リトライ

## 受入基準
- [x] 楽天の商品価格が更新される（税抜）
- [x] himoduke 変換が適用される
- [x] セット商品も更新される
- [x] エラー時にリトライ可能
- [x] licenseKey 期限切れ時に適切なエラーメッセージ

## 注意事項
- 「CSV商品一括登録」オプション（月額10,000円）の契約が必要な場合あり → 要確認
